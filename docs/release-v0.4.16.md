# GamePulse v0.4.16

`v0.4.16` is a protected FPS helper repair hotfix for machines where PresentMon service capture was blocked after install or update.

## Download

- Recommended installer: `GamePulse-Friend-Setup-0.4.16.exe`
- Release page: [GamePulse 0.4.16](https://github.com/ppgnox/gameflipz-native-releases/releases/tag/v0.4.16)

## Highlights

- Restores repair when the protected FPS helper is missing.
- Lets Friend Setup refresh the protected helper by itself.
- Keeps unsafe writable helper elevation blocked.
- Adds helper-refresh evidence to diagnostics exports.
- Reduces noisy PresentMon fallback cleanup log entries.
- Dev-signed release: SmartScreen / unknown-publisher prompts remain expected until production signing is added.
