## Highlights

- Refreshes the public visual story around Compact Mode and the Benchmark overlay.
- Keeps live performance readouts quick to scan in the compact view.
- Keeps Benchmark runs guided around target lock, duration, progress, saved runs, and plain-language verdicts.
- Improves player-facing target readiness so Benchmark starts from a live detected game.
- Expands reviewed game matching for more current PC titles.
- Preserves the external telemetry boundary: no overlays, injection, hooks, game memory access, or game-file changes.
- Dev-signed release: SmartScreen / unknown-publisher prompts are still expected.
