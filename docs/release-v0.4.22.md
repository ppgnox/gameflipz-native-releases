# GamePulse v0.4.22

`v0.4.22` introduces the Benchmark page for guided baseline-to-comparison performance runs, one combined saved report, and player-readable performance context.

## Download

- Recommended installer: `GamePulse-Friend-Setup-0.4.22.exe`
- Release page: [GamePulse 0.4.22](https://github.com/ppgnox/gameflipz-native-releases/releases/tag/v0.4.22)

## Highlights

- Adds guided Benchmark run readiness, target lock, live progress, and saved-run review.
- Saves baseline and comparison captures together so setting changes are easier to judge.
- Explains average FPS, low-FPS evidence, frame consistency, and CPU/GPU pressure in player-facing language.
- Helps players compare changes over time without adding overlays, hooks, game memory access, or game-file changes.
- Keeps benchmark history and public leaderboard sharing under player control.
- Includes the `v0.4.21` FPS capture cleanup after game close.
- Keeps the `v0.4.20` direct update selection fix and the `v0.4.19` friendly FPS repair prompt.
- Preserves prior privacy, auth, public-data, feedback, and installer hardening.
- Dev-signed release: SmartScreen / unknown-publisher prompts remain expected until production signing is added.
