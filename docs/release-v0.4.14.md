# GamePulse v0.4.14

`v0.4.14` focuses on FPS capture repair and Discord/auth plus public-data boundary hardening while keeping the app workflow and native layout unchanged.

## Download

- Recommended installer: `GamePulse-Friend-Setup-0.4.14.exe`
- Release page: [GamePulse 0.4.14](https://github.com/ppgnox/gameflipz-native-releases/releases/tag/v0.4.14)

## Highlights

- Improves PresentMon Service repair for broken FPS capture cases after install or update.
- Reports elevated FPS helper results more clearly when capture setup needs repair.
- Hardens Discord auth and session boundaries without changing the sign-in workflow.
- Redacts more secret-shaped text from public feedback and public-data paths before publication.
- Dev-signed release: SmartScreen / unknown-publisher prompts remain expected until production signing is added.
